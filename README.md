# 📋 Sistema de Notas Médicas NOM-024 (México)

Aplicación web PWA-ready para generar notas médicas de consulta, ingreso y egreso, cumpliendo con la NOM-024-SSA3-2010 y NOM-004-SSA3-2012.

## Características
- Modo oscuro/claro
- Almacenamiento local IndexedDB (pacientes y respaldos)
- Membrete personalizable
- Generación de PDF profesional
- Vistas previas en tiempo real
- Exportar/importar base de datos

## Uso
Abre el archivo `index.html` en cualquier navegador moderno o despliega en GitHub Pages.
